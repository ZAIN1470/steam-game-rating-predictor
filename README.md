# Steam Game Rating Predictor 🎮

## 📌 About
A machine learning project that predicts whether a Steam game 
will be rated positively or negatively based on game features.

## 📁 Files
- `Game_Rating_Predictor.ipynb` - Main notebook with full analysis
- `steam.csv` - Steam games dataset (27,000+ games)

## 🛠️ Tools Used
- Python
- Scikit-learn
- XGBoost
- Pandas
- Matplotlib
- Seaborn

## 📊 Results
| Model | Accuracy |
|-------|----------|
| XGBoost | 65.63% |
| Random Forest | 64.01% |
| Logistic Regression | 62.61% |

## 🔍 Features Used
- Game price
- Average playtime
- Genres
- Platforms
- Categories
- Release date
- Owners count
